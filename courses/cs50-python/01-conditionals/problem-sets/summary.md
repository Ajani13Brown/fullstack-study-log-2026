# Week 1 – Conditionals

**Focus:**  
Control flow using `if`, `elif`, `else`, logical operators, and pattern matching  

**Key Takeaways:**  
- `if` vs `if/elif/else` controls execution flow and prevents unintended outputs  
- Simplifying conditional logic improves readability and maintainability  
- Choosing the right structure depends on the problem (`match` vs `if/elif`)  
- Introduced `match` for handling multiple exact values  
- Functions help organize logic and simplify condition evaluation
