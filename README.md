## Repository Structure

This repository is organized by course, with each course structured into topic-based modules.

```plaintext
fullstack-study-log-2026/
│
├── courses/
│   └── cs50-python/
│       └── 00_variables_and_functions/
│           ├── problem_sets/
│           ├── notes/
│           └── summary.md
│
├── data-structures-and-algorithms/
│
└── progress.md
```

### Courses

Course content is organized into topic-based modules that group related concepts and exercises together.

Within each module:

- `problem_sets/` → completed exercises and solutions  
- `notes/` → lecture notes and deeper exploration  
- `lecture_*` → follow-along notes from course material  
- `notes_*` → additional insights and deeper exploration  
- `summary.md` → concise weekly summary highlighting key takeaways and understanding  

### Progress Tracking

Progress is tracked in `progress.md`, which provides a high-level view of current focus, active work, and completed milestones.

It is designed to clearly communicate direction, consistency, and measurable progress over time.
